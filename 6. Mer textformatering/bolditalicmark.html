What is Java?
Java is an object-oriented programming language created by Sun Microsystems Company. It can operate on any type ("virtual machine") of operating system, or computer system ( PC, Mac, Solaris, etc.). It is a flexible tool for integrating interactivity and multimedia, and a very secure language, making it a useful language for networking or the Internet.

What is JavaScript?
JavaScript is a powerful scripting language created by Netscape. It is primarily used to add interactivity and multimedia to web pages.
